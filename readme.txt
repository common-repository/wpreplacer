== Description ==
wpReplacer is a post rewrite plugin, it can automatic rewrite your posts or pages

you can setting:

Replace Level:
Good replace the most terms
Better fewer replacements, higher quality
Best fewest replacements, higher quality

the percentage of replaceable words you would like to rewrite

Negative Words: words that will not be replaced


== Installation ==

1. Upload the directory `wpreplacer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
